Learn 10+ Essential UNIX Commands

For the learning Terminal commands, let's load the HelloWorld.activity from the examples window on the project tab.

We can use the Terminal built into PyDebug. Click on the "Activity" tab at the upper left. Then type <control>D. You should see "[HelloWorld.activity]$". This is the terminal prompt, and tells you that the Terminal is ready to listen to your commands.

Here is a quick list of the commands we will be using in this lesson (there is also a more complete list of commands):

  1. ls -- list contents of current directory
  2. cd -- change directory
  3. pwd -- print working directory
  4. mkdir - make a new directory (rmdir - remove directory)
  5. cp -- copy file or directory
  6. rm - remove file or directory
  7. df -- disk free
  8. du -- disk used
  9. whoami -- who am I?
  10. su -- super user
  11. less -- improved version of "more" -- the original unix page viewer
  12. vi -- visual text editor -- available and installed by default on all unix machines

Files and Folders

The Journal of the XO is a simplified view of the file system. It is a single container, or folder, that contains ONLY files. The Journal shows the most recent things on top. The order is based upon time. This simplified view of the file system hides a lot of what is actually stored on your XO.

In actual fact, Folders can contain other folders, as well as files. Often the complication comes when you can't remember which folder you put something in. It's there somewhere, but you can't find it. Your files will be easier to find if you learn the rules, where things are usually kept in a unix computer.

The first part of this lesson asks you to become familiar with some of the folders and files on your XO.
